# homework01

這是一個「倍元素週期表」靜態網站專案。網站已包含以下檔案：

- `index.html`：網站主頁
- `style.css`：版面樣式
- `script.js`：互動邏輯

## 使用方式

1. 打開 `homework01/index.html`，直接用瀏覽器檢視網站。
2. 或者在終端機中進入 `homework01` 資料夾，執行：

```bash
python3 -m http.server 8000
```

然後在瀏覽器打開：

```text
http://127.0.0.1:8000
```

在網頁上輸入一個數字，就會高亮所有原子序數為該數字倍數的元素。
